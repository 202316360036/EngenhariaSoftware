# Questionário para Entrevistas com Stakeholders — MotoJá

## Grupo A — Negócio (Donos/Investidores)
1. Como o MotoJá vai monetizar? (% por corrida, mensalidade, ambos?)
2. Existe tarifa mínima? Qual?
3. Qual área do piloto (cidade/bairros) e limites de atuação?
4. Regras de cancelamento: quando cobra taxa do passageiro?
5. Quais KPIs são prioridade no painel admin (corridas/dia, motoristas online, faturamento, cancelamentos)?
6. Quais critérios para aprovar/reprovar mototaxistas?

## Grupo B — Mototaxistas (Usuário final)
1. Você prefere ver valor antes de aceitar ou só destino?
2. O que mais atrapalha aceitar rápido (chuva, luvas, tela)?
3. Quais motivos mais comuns para recusar corrida?
4. Quais infos são essenciais no aceite (valor, distância, destino, tempo)?
5. O que ocorre na prática quando a internet cai durante a corrida?
6. Qual forma de pagamento é mais comum (Pix/cartão/dinheiro)?

## Grupo C — Passageiros (Usuário final)
1. Você quer ver preço estimado antes de confirmar?
2. Qual forma de pagamento prefere (Pix/cartão/dinheiro)?
3. Em quais casos você cancelaria a corrida?
4. O que você espera ver no mapa durante a corrida?
5. Quais dados aceitaria compartilhar (GPS/telefone)?
6. O que seria uma experiência ruim que o app deve evitar?
